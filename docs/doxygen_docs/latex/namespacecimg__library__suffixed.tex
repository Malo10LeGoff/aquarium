\doxysubsection{Detailed Description}
Contains {\itshape all classes and functions} of the \textbackslash{}\+C\+Img library. 

This namespace is defined to avoid functions and class names collisions that could happen with the inclusion of other C++ header files. Anyway, it should not happen often and you should reasonably start most of your \textbackslash{}\+C\+Img-\/based programs with 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include "\mbox{\hyperlink{CImg_8h}{CImg.h}}"}}
\DoxyCodeLine{\textcolor{keyword}{using namespace }cimg\_library;}
\end{DoxyCode}
 to simplify the declaration of \textbackslash{}\+C\+Img Library objects afterwards.
